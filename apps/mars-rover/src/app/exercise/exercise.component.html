Exercise

  1. Create a new page "/marsrover".
    • The page should show the component "MarsRoverComponent".
    • Provide some simple help for navigation.
  2. The page "/marsrover" shows a landscape and the mars rover.
    • Add commands (turn, move) to the page, to be able to control the movements.
  3. The mars rover should analyse rocks. While analyzing it can't move or turn, but the commands are queued and processed afterwards.
    • Analyzing takes 3 seconds.
